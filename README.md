## Description
DividentDiary is a mobile Android application that allows users to calculate Unit Trust dividend earnings using standard formulas based on investment amount, annual dividend rate, and duration in months (up to 12).

It provides accurate, clean, and mobile-friendly UI with full calculations and outputs formatted to two decimal points.

---

## Features:
- Accurate dividend calculation
- Monthly + total output
- Input validation
- Navigation drawer (Home / About / GitHub / Exit)
- Clean green UI theme
- Clickable GitHub repository link
